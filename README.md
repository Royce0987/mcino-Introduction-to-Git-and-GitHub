# Simple Interest Calculator

A calculator that calculates simple interest given principal, annual rate of interest and time period in years.

## Introduction

This is a simple interest calculator built using a Bash script. It takes user input for principal, rate of interest, and time period, then computes and displays the simple interest.

## Inputs
- p, principal amount
- r, annual rate of interest  
- t, time period in years

## Output
- Simple Interest = p*t*r/100

## Usage

Run the script:
```bash
bash simple-interest.sh
```
